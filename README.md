# python-django-blog

[Demo on Heroku](https://python-django-blog.hexlet.app)

## Requirements

* Python 3.8+
* Poetry
* GNU Make

## Setup

```bash
make setup
```

## Run server

```bash
make start
# Open http://localhost:8000
```

## Check codestyle

```bash
make lint
```

## Run tests

```bash
make test
make test-coverage # run tests with coverage report
```
